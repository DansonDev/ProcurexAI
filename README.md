# 📦 IS for Procurement Automation & Supply Analytics

## 🧾 Описание проекта
Информационная система для автоматизации закупок и анализа поставок в торговой компании.

Система анализирует продажи и остатки на складе, а также автоматически формирует закупки на основе аналитики.

---

## 🎯 Цель проекта
Разработать приложение, которое:
- анализирует продажи
- отслеживает складские остатки
- автоматически формирует закупки при необходимости
- использует аналитику и ML (в перспективе)

---

## ✅ Критерии успеха
- 📊 Управление поставщиками  
- 📦 Управление товарами  
- 🛒 Закупки (на основе аналитики)  
- 🚚 Автоматизация поставок  
- 🤖 Аналитика (ML)
